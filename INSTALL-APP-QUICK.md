# 📱 Install TaskFlow as Mobile App - Quick Guide

## 🍎 iPhone/iPad (iOS)

1. Open **Safari** browser
2. Go to: `https://abdallared.github.io/task-management-system/`
3. Tap the **Share** button (⬆️) at the bottom
4. Scroll and tap **"Add to Home Screen"**
5. Tap **"Add"**
6. Done! 🎉 The app icon is now on your home screen

```
Safari → Share (⬆️) → Add to Home Screen → Add
```

---

## 🤖 Android

1. Open **Chrome** browser
2. Go to: `https://abdallared.github.io/task-management-system/`
3. Tap **"Install"** banner at the bottom
   - OR tap menu (⋮) → "Install app"
4. Tap **"Install"**
5. Done! 🎉 The app icon is now on your home screen

```
Chrome → Install banner → Install
```

**iOS:** Must use Safari browser (not Chrome)

- Refresh the page
- Make sure you're on the actual website
- Check if already installed
