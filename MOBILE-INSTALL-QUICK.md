# 📱 Mobile Install - Quick Reference

## 🤖 Android (Chrome)

### Quick Steps:
1. Open **Chrome** on Android
2. Go to: https://abdallared.github.io/task-management-system/
3. **Interact with the site for 30+ seconds** (scroll, click, login)
4. Tap **menu (⋮)** → **"Install app"**
5. Tap **"Install"**
6. Done! ✅

### Not seeing "Install app"?
- Clear Chrome data: Settings → Apps → Chrome → Storage → Clear data
- Restart phone
- Update Chrome from Play Store
- Interact with site for 30+ seconds before checking menu

---

## 🍎 iOS (Safari)

### Quick Steps:
1. Open **Safari** (must be Safari, not Chrome!)
2. Go to: https://abdallared.github.io/task-management-system/
3. Tap **Share button (⬆️)** at the bottom
4. Scroll down and tap **"Add to Home Screen"**
5. Tap **"Add"**
6. Done! ✅

### Not seeing "Add to Home Screen"?
- Make sure you're using Safari (not Chrome)
- Exit private browsing mode
- Clear Safari: Settings → Safari → Clear History and Website Data
- Restart iPhone/iPad

---

## ⚠️ Important Notes

### Android:
- Must use **Chrome browser**
- Must **interact with site for 30+ seconds** before install option appears
- Look for **"Install app"** in menu (not "Add to Home screen")
- If only see "Add to Home screen", that's a bookmark (not PWA)

### iOS:
- Must use **Safari browser** (Chrome on iOS doesn't support PWA install)
- **No automatic prompt** - must use Share → Add to Home Screen
- This is the ONLY way to install on iOS
- Works on iOS 11.3 or later

---

## 🔍 Quick Troubleshooting

**Android - "Install app" not in menu:**
1. Clear Chrome data (not just cache)
2. Restart phone
3. Visit site and interact for 30+ seconds
4. Check menu again

**iOS - "Add to Home Screen" not in Share menu:**
1. Confirm using Safari (not Chrome)
2. Exit private browsing
3. Clear Safari data
4. Restart device
